# Changelog

## 1.4.0 - 2026-05-03

### New features

* Added a `DefaultConfig` option to provide a way to deploy a config
  if one doesn't exist.

### Changes

* Dependency updates

## 1.3.0 - 2025-07-04

### Changes

* Switched YAML parser library to a maintained version.

## 1.2.0 - 2021-07-04

### Changes

* Added Directory() method to expose the config folder, so other files can be
  stored there.

## 1.1.1 - 2021-07-04

### Changes

* Only use the last element of `argv[0]` to determine the name of the
  config directory.

## 1.1.0 - 2021-10-28

### Changes

* Added package-level Load func.

## 1.0.0 - 2021-07-04

_Initial release._
